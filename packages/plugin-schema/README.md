## 使用

### 注册插件
```jsx
import { plugins } from '@digiforce-cloud/dvd-engine';
import LowcodePluginAliLowcodePluginSchema from '@digiforce-cloud/dvd-plugin-schema';

// 注册到引擎
plugins.register(LowcodePluginAliLowcodePluginSchema);
```
